# Weltgeschichte der Klassenkämpfe (1914–2026)

Eine **dialektisch-materialistische Analyse** der globalen Klassenkämpfe.
Basierend auf den Werken von **Marx, Lenin und Mao**.

### Struktur
- [docs/specs/](docs/specs/) – Spec-Dateien (YAML) für Kapitel
- [docs/bands/](docs/bands/) – Generierte Markdown-Kapitel
- [src/data/](src/data/) – Datenbank der Klassenkämpfe (CSV)
- [.github/workflows/](.github/workflows/) – Automatisierung (CI/CD)
